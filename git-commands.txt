# Команды для загрузки проекта в GitHub

# 1. Инициализация Git репозитория (если еще не сделано)
git init

# 2. Добавление всех файлов
git add .

# 3. Первый коммит
git commit -m "Initial commit: Система автоматического обновления плагинов Revit

- Серверная часть на ASP.NET Core 8
- Веб-админка на JavaScript + Bootstrap
- Клиент для Revit на .NET Framework 4.8
- Утилита updater.exe для замены файлов
- Конфигурация для развертывания на Render.com
- Полная документация по использованию и развертыванию"

# 4. Подключение к удаленному репозиторию
git remote add origin https://github.com/JessiePinkMaan/revit-plugin-updater.git

# 5. Отправка кода в GitHub
git branch -M main
git push -u origin main

# Если возникнут проблемы с аутентификацией, используйте Personal Access Token:
# 1. Перейдите в GitHub Settings > Developer settings > Personal access tokens
# 2. Создайте новый token с правами repo
# 3. Используйте token вместо пароля при push
